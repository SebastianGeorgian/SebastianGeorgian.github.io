mygit — Git, from scratch

systems programming · C11

A working subset of Git in C11 with no external libraries — init · add · commit · log · status · checkout. Not a wrapper around the real thing: it implements the same model — a content-addressable object store keyed by SHA-256 (also written from scratch), a binary staging area, and refs — so checkout genuinely rebuilds any snapshot from hashes alone.

6 commands ~1.4k lines of C 0 dependencies SHA-256 hand-written 14 test scenarios ASan + UBSan clean
